# ProgressTeaching App

## Getting Started

First, create an environment file:

```bash
cp .env.example .env
```

Install packages

```bash
npm install
```

## CLI

### Running tests

Run once:

```bash
npm test
```

Watch:

```bash
npm test:watch
```

Run CI:

```bash
npm run test:ci
```

### Prettier Check

```bash
npx prettier --check .
```

### Prettier fix

```bash
npx prettier --write .
```
